# create-react-repo-app
A cli to create react project quickly... 

# install
```sh
    npm install create-react-repo-app -g

    // or
    yarn add create-react-repo-app -global
```
```sh
    create-react-repo-app init [project-name]
```
```sh
    cd [component-name]
```
```sh
    npm i 

    // or
    yarn 
```
